-Sign In Page
   This project consists of a simple sign-up and sign-in page created using HTML and CSS. The    purpose of this project is to provide a user-friendly interface for users to register for an account    or log in to an existing one.
-Features
   Sign-Up Page: Users can enter their information such as username, email, and password to    create a new account.
   Sign-In Page: Existing users can log in using their credentials.
   Responsive Design: The pages are designed to adapt to various screen sizes, ensuring usability    across devices.
   CSS Styling: The pages are styled using CSS to enhance visual appeal and user experience.
